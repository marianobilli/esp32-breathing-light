How the pins of the Oled Board with controls connect to the ESP32 pins

## ESP32 PIN : OLED Board PIN

4: CON (Confirm Button)
5: SDA
3: SCL
8: PSH (Rotary control PUSH button)
9: TRA (Rotary control A)
18: TRB (Rotary control B)
17: BAK (Back button)

ESP32-S3 GPIO: MAX98357A Pin │ Notes

---

3v3: VCC
GND: GND
41: BCLK │ I2S Bit Clock
42: LRC │ I2S Word Select
40: DIN │ I2S Serial Data
GND: SD │ Shutdown (tied to GND, always on)
floating: GAIN │ 9 dB fixed gain
Speaker: SPK+ / SPK- │ 4Ω or 8Ω
